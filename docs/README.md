## Quick Start

```bash
# 1. Clone & setup environment
git clone <repo-url>
cd blender_mcp_project
setup_and_index.bat

# 2. Install the Blender addon
install_bridge.bat

# 3. Enable in Blender: Edit > Preferences > Add-ons > "Elite MCP Bridge v2"

# 4. Inject MCP into your IDE
install_gemini_mcp.bat

# 5. Start using 54 structured tools through your AI assistant!
```
